O Que Somos?

O projeto Morea tem como objetivo realizar estudos sobre a aplicação, implementação e uso de um protótipo de sistema integrado para o monitoramento setorizado do consumo de água, energia e qualidade do ar, permitindo a identificação de situações atípicas, além de prever estatísticas mensais dos mesmos dados. {{ data }}

Dashboard

Um dashboard é um painel visual que tem função de apresentar informações de maneira dinâmica e objetiva, como também possibilitar uma visão geral dos dados específicos do projeto. Assim, gerando a tendência de facilitar a correção de possíveis erros e pensar em novas soluções para a inovação do produto apresentado.

Dados Brutos

Esta é uma página dedicada a visualização dos dados em sua forma bruta, afim de fornecer maior controle na análise tanto de consumo quanto para identificar possíveis vazamentos no sistema, sendo crucial para agilizar diversas terefas para a equipe.
